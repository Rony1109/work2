深圳商友线下见面会

  • 活动形式：线下活动
  • 开始时间：012年10月30日 08:25
  • 结束时间：2012年11月30日 18:00
  • 活动发起：深圳和顺发纺织有限公司

我要评论

己报名：16人   我要报名

活动描述:

阿里博客培训班，到目前为止，已经举办了四期。前几期都是以线上活动为主的，这一期开始，在线上活动的同时也开始启动了线下活动，因为深圳算是比较集中的一个区域，由黄欣子来负责本次深圳分站的线下活动，本次活动主题为博客营销的分享和集体烧烤两部分组成， 既要学东西，也要填饱肚子的，搞烧烤也是比较有意思的，到时候看看这里面谁是烧烤技术最高的哈？大家可以拿起你的相机，手机记录下这精彩的一瞬间。

活动描述:

活动地址：深圳福永凤凰山（山脚下那块石头前面的广场集合）
活动时间：2012年10月21日上午11点（因为部分人比较远，为了不影响大家周末睡懒觉）到下午3点结束（据说晚上博客培训班还有课程，为了不耽误大家听课，尽早结束）

活动注意事项：

1．请大家自带一些零食，矿泉水有同学赞助，可以不带。此外还记得带好笔，要填一份调查问卷。
2.烧烤的东西我们会提前买好，带过去。
3.本地活动费用AA制，先暂定每人100元，带家属的话，按人头算，请把钱统一交给我们的纪委刘蓉书记，活动结束以后，按实际支出多退少补。
