产品详细信息

一、 简介 棉花是巴基斯坦重要的经济作物，也是纺织行业的重要原材料。棉花行业在农业部门的附加值中占8.2%，为全国GDP贡献大约28亿美元，占总量大约2%，据估计，棉花产量每增加100万包（170公斤/包），将为全国GDP增加0.5%。棉花和棉制品在全国GDP中占10%，并占全国出口创汇的55%。总体而言，巴基斯坦国内棉花产量中大约30-40%被生产成最终产品用于国内消费，剩余棉花则以棉花、棉纱或者棉织品的形式出口。 目前，巴基斯坦棉花行业在发展中主要受到以下限制：  棉花单产低，棉种、化肥和农用等农业投入高；  在棉花生长初期、花期和铃期害虫影响较大；  高质量的高产抗虫棉棉种供应紧张；  灌溉设施有限；  缺乏先进的科学技术；  缺乏农业技术和培训人员；  农药、化肥和棉种供应不足。 除以上因素之外，降雨、干旱和洪水等自然因素也将对全国尤其是旁遮普和信德省的棉花产量带来很大影响。棉花种植期间如果遭受不利天气，将影响出苗，需要补种，很有可能对单产带来一定影响。 由于棉花产量低，因此巴基斯坦每年需要进口150-200万包棉花以满足日益增长的纺织用棉需求。另外，由于棉花质量不高，因此巴基斯坦棉花产值因此大约减少10-15%。为了解决这个问题，扩大棉花种植面积可能是方案之一。然而，目前国内两个主要产棉区旁遮普和信德省的棉花面积已基本接近最大值，因此继续增加的空间不是很大。另外一个解决方案就是采用现代棉花技术，比如从发达国家和其他发展中国家引进优良的棉花品种，以便提高单产，增强抗虫能力等等。
